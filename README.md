# @verdaccio/eslint-config

eslint shareable configuration for Verdaccio that uses Typescript.

```
 npm install -D @verdaccio/eslint-config
```

## Usage

```
// .eslintrc
{
  "extends": [
    "@verdaccio"
  ]
}
```
